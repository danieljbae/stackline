## Installation

1. Clone the repository
```sh
git clone https://github.com/danieljbae/stackline.git
```

1. Install NPM packages
```sh
npm i
```

1. Start the project
```sh
npm start
```

## Deployment 
To build and deploy this project on GitHub Pages, run:
```sh
npm run build
npm run deploy
```
